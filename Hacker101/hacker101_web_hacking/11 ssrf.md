# Server-Side Request Forgery

In this session we’ll talk about server-side request forgery. This is when an attacker controls the target of HTTP(S) requests coming from the server.

## What you’ll learn

  * SSRF 
    * File accesses
    * Firewall bypass
    * Port scanning
    * Mitigation
